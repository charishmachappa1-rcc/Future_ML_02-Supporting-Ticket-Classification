# 🚀 Support Ticket Classification System (NLP)

## 📌 Overview
This project builds an NLP-based system to automatically classify customer support tickets and assign priority levels (High, Medium, Low).

It helps support teams respond faster and manage workload efficiently.

---

## 🎯 Problem Statement
Customer support teams receive a large number of tickets daily. Manually categorizing and prioritizing them is time-consuming and inefficient.

This project automates:
- Ticket classification
- Priority assignment

---

## ⚙️ Features
✅ Text cleaning & preprocessing  
✅ Tokenization using NLP  
✅ Ticket category classification  
✅ Priority prediction (High / Medium / Low)  
✅ Model performance evaluation  

---

## 🛠️ Tech Stack
- Python 🐍  
- NLTK / spaCy  
- Scikit-learn  
- Pandas / NumPy
